Modelo Avançado de Previsão de Desemprego

Autor: Luiz Tiago Wilcke

1. Introdução e Explicação

Este modelo utiliza uma regressão autoregressiva distribuída (ARDL) para prever a taxa de desemprego com base em variáveis macroeconômicas-chave.

As variáveis explicativas consideradas são:

  • \(\text{GDPG}\): Gap do Produto Interno Bruto (%)
  • \(\pi\): Taxa de inflação anual (%)
  • \(i\): Taxa básica de juros anual (%)
  • \(\Delta LF\): Crescimento da força de trabalho (%)
  • \(U_{t-1}\): Taxa de desemprego do período anterior (%)

2. Modelo Matemático

\[ U_t = \alpha + \beta_1 \mathrm{GDPG}_t + \beta_2 \pi_t + \beta_3 i_t + \beta_4 \Delta LF_t + \beta_5 U_{t-1} + \varepsilon_t \]

Onde:

  • \(U_t\): Taxa de desemprego prevista no período \(t\).
  • \(\alpha, \beta_i\): Coeficientes estimados via regressão.
  • \(\varepsilon_t\): Termo de erro.
