Modelo: \[ \pi_t = \beta_0 + \beta_1 \pi_{t-1} + \beta_2 r_{t-1} + \beta_3 e_{t-1} + \varepsilon_t \]

--

Como o Modelo Funciona

Este modelo linear múltiplo prevê a inflação atual (πₜ) com base em quatro componentes principais:

  1. Intercepto (β₀): valor de referência da inflação quando todas as variáveis independentes são zero.
  2. Inflação defasada (πₜ₋₁): representa a inércia da inflação—isto é, como o valor passado impacta o presente. O coeficiente β₁ mede essa persistência.
  3. Taxa Selic defasada (rₜ₋₁): a política monetária anterior influencia a inflação atual. O coeficiente β₂ mostra essa sensibilidade.
  4. Câmbio defasado (eₜ₋₁): variações no câmbio passado afetam preços de bens importados. O coeficiente β₃ quantifica esse efeito.

A equação matemática do modelo é:

\[ \pi_t = \beta_0 + \beta_1 \pi_{t-1} + \beta_2 r_{t-1} + \beta_3 e_{t-1} \]

Para usar, insira os valores nos campos acima e clique em Calcular Inflação. O resultado exibirá a previsão de inflação (πₜ) em porcentagem.
